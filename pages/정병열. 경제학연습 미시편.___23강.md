	- $${노동소득 \over 자본소득} = {wL \over rK} = {w \over r}{L \over K} = {MP_L \over MP_K}{L \over K}$$
	- $$\sigma = {{K \over L} 변화율 \over {w \over r} 변화율} = {{K \over L} 변화율 \over MRTS변화율}$$
	- 선형생산함수 $$\sigma = \infty$$
	- 레온티에프 생산함수 $$\sigma = 0$$
	- 콥-더글라스 생산함수 $$\sigma = 1$$
